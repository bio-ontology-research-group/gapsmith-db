# Architecture Decision Records

Brief ADRs for any non-obvious design choice in gapsmith-db. Numbered
sequentially. Use the template below.

```markdown
# ADR NNNN: Short title

- **Status**: proposed | accepted | superseded by NNNN | deprecated
- **Date**: YYYY-MM-DD
- **Phase**: 0..5

## Context
What forces are at play?

## Decision
What did we choose?

## Consequences
What becomes easier, harder, or load-bearing because of this?

## Alternatives considered
What else was on the table and why was it rejected?
```
